Detailed Description

The GRTextToSpeech class controls the sound settings and converts text to speech.

The GRTextToSpeech class controlls the different sound parameters such as the sound volume, rate and pitch. It also controls the process of converting text to speech such as the language, the voice (speaking person) and the conversion engine.
